# Library Management System (Java Console Application)

A simple **Library Management System** developed using **Core Java**.  
This application allows administrators to manage books and members, while users can borrow and return books through a console-based interface.

The project demonstrates the use of:
- Object-Oriented Programming (OOP)
- Collections Framework
- User authentication
- Basic system design logic

---

## Project Overview

This system simulates a real-world library environment where:

Admin can:
- Add new books
- Update book details
- Remove books
- Add new members
- View all books
- View all members

User can:
- Borrow books
- Return books
- View available books

Each member can borrow up to **5 books** at a time.
